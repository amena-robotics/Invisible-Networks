# Hardware specs

Detailed hardware specifications, wiring and assembly

---

## Materials and components

| Part | Spec | Notes |
|------|---------------|-------|
| **Electronics** | | |
| Raspberry Pi Zero 2W | ARMv7, 512MB RAM, WiFi/BT | Microcontroller |
| Raspberry Pi Camera Module | 5MP, 160° FOV | Vision system |
| L293D Motor Driver | Dual H-bridge, 600mA max | Motor control |
| Boost Converter | 3.2V→5V or 3.7V→5V | Power regulation |
| **Power** | | |
| LiFePO4 Battery | 3.2V, 1000mAh | Primary power (safer) |
| OR LiPO Battery | 3.7V, 1000mAh | Alternative (monitor for safety) |
| Battery Connector | XT60 or JST-PH | Depends on battery |
| **Sensors & outputs** | | |
| Water Level Sensor | Capacitive or resistive | Detects submersion |
| Status LED | 3-5V, 2-3 units | Power/status indicators |
| LED Resistor | 220Ω, 0.25W, 2-3 units | Current limiting |
| **Actuators** | | |
| Electromagnetic Coil | Hand-wound, 10-20Ω | Tail propulsion |
| Mechanical Coupler | Flexible, 3mm shaft | Motor to coil connection |
| **Frame & waterproofing** | | |
| Silicone (for casting) | Translucent silicone | Waterproof body |
| 3D Printed mould | PLA | Fish-shaped frame |
| O-rings | Silicone, various sizes | Waterproof seals |
